# CODSOFT Internship Repository

This repository contains all tasks completed during my CodSoft C++ Programming Internship.  
Each task demonstrates practical implementation of core programming concepts.

---

## 📂 Tasks

### Task 1: Number Guessing Game
- Generates a random number
- User guesses until correct
- Provides feedback: Too high / Too low

### Task 2: Simple Calculator
- Performs addition, subtraction, multiplication, division
- User inputs two numbers and selects operation
- Handles invalid choices gracefully

### Task 3: Tic-Tac-Toe Game
- Console-based 3x3 grid
- Two players: X and O
- Checks win/draw conditions
- Replay option included

### Task 4: To-Do List Manager
- Add, view, update, and delete tasks
- Default status: "PENDING"
- Allows marking tasks as completed
- Menu-driven interface

### Task 5: Library Management System 
- Manage books, borrowers, and transactions
- Search, checkout, return, fine calculation
- User-friendly console interface
